**Tech Stack:** Laravel • ReactJS • TailwindCSS • PostgreSQL

### Contract & Legal AI Reasoning Assistant

Built an AI-powered platform to assist MSMEs with legal licensing information and automated document generation using modern LLM technologies.

Integrated AI tools including LlamaIndex, Gemini, and OCR systems to deliver intelligent reasoning and automation features.

🏆 Achieved **2nd Place Winner** at the Arkavidia National Hackathon.

**Tech Stack:** Express.js • React • LLM • Neo4J
